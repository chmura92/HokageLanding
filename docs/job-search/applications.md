## How to use

1. **Pipeline table** — one row per applied role. Edit inline as status changes. Commit each change (`git commit -m "track: Volvo → screening"`).
2. **New row** — append at bottom with next number. Required: Date, Company, Role, Channel, Link, Status.
3. **Deep notes** — per-company sections below the table (#1 – #15 pre-populated). Append new `### N. Company` sections as you add rows.
4. **Weekly review** — every Friday, scan the table, pick 1-2 `💤 Ghosted` for follow-up, update `Next action` column.

## Status legend

| Emoji | Status | Meaning |
|---|---|---|
| 📋 | Todo | Identified, not applied yet |
| 📤 | Applied | Submitted, no response yet |
| 👀 | HR review | Recruiter acknowledged receipt |
| 📞 | Screening | Phone/video screen scheduled or done |
| 🗓️ | Interview | Hiring manager / team interview |
| 🧪 | Technical | Tech interview / take-home task |
| 💰 | Offer | Offer on the table |
| ✅ | Accepted | Signed |
| ❌ | Rejected | They said no |
| 🚫 | Declined | I said no |
| 💤 | Ghosted | >21 days silence |

---

## Rhythm rules

- **5 applications / week max.** Quality over quantity.
- **Tue/Wed/Thu, 10:00-14:00** — best submit window.
- **No Mon morning, no Fri afternoon.**
- Custom pitch per application — change min 2 sentences from base.
- After submit: LinkedIn follow-up to a recruiter from that company within 24h.
- If `💤 Ghosted` after 7 days — one polite LinkedIn nudge. After 14 days — move on.
